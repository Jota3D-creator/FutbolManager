# Los Bosteros · Fútbol 7 Manager

Subí estos archivos al repo:

- index.html
- style.css
- app.js

Contraseña de editor inicial: `bosteros2026`

Podés cambiarla en `app.js`:

```js
var EDITOR_PASSWORD = "bosteros2026";
```

La contraseña es un bloqueo práctico, no seguridad real fuerte, porque GitHub Pages es estático.

Funciones:
- Dashboard con próximo miércoles, Benimàmet CF, costo 5€ y mapa.
- Modo visitante/editor.
- Agregar/editar jugadores.
- Fotos por jugador.
- Confirmado / En duda / No viene.
- Generar equipos.
- Ver equipos sobre una cancha.
